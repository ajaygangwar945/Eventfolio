## ‚ú® Visual Identity

The project follows a "Cyber-Ghibli" design philosophy:

- **Watercolor Foundations**: Soft, hand-painted backgrounds create a warm, nostalgic atmosphere.
- **Floating Lanterns & Spirits**: Subtle CSS animations that breathe life into every page.
- **Glassmorphism**: UI elements use frosted glass effects with smart backdrop filtering for modern legibility.
- **Parallax Vistas**: Multi-layered backgrounds that react to scroll, creating depth and immersion.
